⚠️ Important Disclaimer

This is a DEMONSTRATION extension

PopFact uses simple keyword matching, NOT real fact-checking.

DO NOT rely on these results for accurate information.

What PopFact Does:

  • Displays a CNN-style news ticker at the bottom of web pages
  • Extracts text content from pages you visit
  • Matches claims against simple keyword patterns
  • Shows mock results in the ticker (for demo purposes)

What PopFact Does NOT Do:

  • Connect to real fact-checking databases
  • Provide accurate or reliable verification
  • Use AI or machine learning for analysis
  • Transmit your data to external servers
How It Works:

The extension uses basic keyword detection (e.g., "climate change" → TRUE, "flat earth" → FALSE). All processing happens locally in your browser. This is a proof-of-concept demonstration only.

For Real Fact-Checking, Visit:

  • Snopes.com - Fact-checking since 1994
  • FactCheck.org - Nonpartisan fact-checking
  • PolitiFact.com - Pulitzer Prize winning fact-checking
  • Reuters Fact Check - Global fact-checking
